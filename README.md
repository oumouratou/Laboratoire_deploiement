# Laboratoire_deploiement

Dépôt préparé pour héberger séparément les trois services du projet.

## Arborescence cible

- `spring-boot/` : application Spring Boot
- `vue-js/` : application Vue.js
- `flask/` : API Flask

## Source actuelle à déplacer

- Spring Boot : `Backend_deployement/backend_labo2`
- Vue.js : `Frontend2 -security/Stage_frontend`
- Flask : `Frontend2 -security/Stage_frontend/chatbot-api`

## Hébergement conseillé

- Spring Boot : Render, Railway, Fly.io ou VPS avec Java 17+
- Vue.js : Vercel, Netlify ou hébergement statique
- Flask : Render, Railway ou VPS avec Python 3.11+

## Prochaine étape

Initialiser ce dossier comme nouveau dépôt Git, puis pousser sur GitHub sous le nom `Laboratoire_deploiement`.
